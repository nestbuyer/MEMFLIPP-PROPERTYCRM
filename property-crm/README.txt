GEOVANNY PROPERTY CRM — Netlify deploy
======================================

WHAT'S IN HERE
  index.html             -> the CRM app (this is the site)
  manifest.webmanifest   -> makes "Add to Home Screen" show the app name + icon
  apple-touch-icon.png   -> home screen icon (iPhone)
  icon-512.png           -> app icon (Android / browsers)
  netlify.toml           -> tells Netlify how to serve the site

HOW TO DEPLOY (drag & drop, free)
  1. Unzip this folder onto your computer.
  2. Go to  https://app.netlify.com/drop
  3. Drag the WHOLE "property-crm" folder onto the drop zone.
  4. Wait a few seconds -> you get a live link like  yourname.netlify.app
  5. (Optional) Site settings -> Change site name -> e.g. geovanny-crm

PUT IT ON YOUR PHONE
  - Open the Netlify link in Safari (iPhone) or Chrome (Android)
  - Tap Share -> Add to Home Screen

UPDATING LATER
  - Drag the new folder onto the same site (Deploys tab). The link stays the same.

NOTE ON YOUR DATA
  - Tasks and crew phone numbers save on each device separately.
  - Use "Send to Team -> Download backup file" to save a copy or move data
    to another device (restore with the same button).
